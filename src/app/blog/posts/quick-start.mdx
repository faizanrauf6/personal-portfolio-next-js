## About

My Portfolio is a comprehensive, MDX-based, SEO-friendly, responsive portfolio template built with Once UI and Next.js.

## Quick start

Clone the git repository:

<CodeBlock compact marginBottom="16" codeInstances={[
  {
    code: "git clone https://github.com/faizanrauf6/personal-portfolio-next-js.git",
    language: "bash"
  }
]} />

Install the necessary dependencies:

<CodeBlock compact marginBottom="16" codeInstances={[
  {
    code: "npm install",
    language: "bash"
  }
]} />

Start the local development server:

<CodeBlock compact marginBottom="16" codeInstances={[
  {
    code: "npm run dev",
    language: "bash"
  }
]} />
